# JavaScript

### JavaScript

- 사용자와 상호작용하는 언어
- HTML을 제어하는 언어
- 동적인 언어(JS) ↔ 정적인 언어(HTML)
- 	``
		<script></script>
	``

---

### event
- 속성 값으로 JS가 와야 함
- 속성의 속성 값은 웹브라우저가 기억하고 있다가 실행하면 JS 문법에 따라 해석하여 웹브라우저가 동작함
- ex) onclick()
