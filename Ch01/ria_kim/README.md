#HTML강의요약

>   <strong>a</strong> : 글자 강조
    <u>a</u> : 글자 밑줄
    <h1>a</h1> : heading(h1~h6) 
    <br> : 줄바꿈 태그 (닫는 태그 없음)
    <p>a</p> : 단락 태그

>>        -> <br> vs <p></p>
<p>태그를 사용하는게 단락을 더 확실히 나눌 수 있다

<img src="이미지주소(url)">
: html파일과 같은 위치에 img파일을 넣어주면 src 속성의 값으로 이미지 이름만 적어도 됨.
-> <img src="coding.jpg">
<img width 숫자(%)> : 이미지 크기 조정
==> <img src="coding.jpg" width="100%">

<li></li> : 리스트
<ul></ul> : <li>태그를 그룹핑 하기위한 부모태그 (unordered)
<ol></ol> : <li>태그를 그룹핑 하기위한 부모태그 (ordered)
-> <ul>
			<li></li>
	 </ul>

<title></title> : 웹 페이지 제목
<meta charset="utf-8"> : 글자 깨지지 않도록 하는 태그

<!doctype html>
<html>
<head></head>
<body></body> : 구조


<a href="링크(url)" target="_blank" title="제목">a</a> : 하이퍼링크 (anchor) + 새 탭으로 열기 + 링크 열기 전 제목 보여주기
